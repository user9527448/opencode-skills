# Pre-Refactor Checklist Template

## Code Health

- [ ] I understand what the code does (read it 3x)
- [ ] I understand how it's tested
- [ ] I know all call sites (who uses this code)
- [ ] I have identified all code smells present

## Test Status

- [ ] All tests pass (GREEN)
- [ ] Test coverage > 70% (or manually verified)
- [ ] No flaky tests
- [ ] I know which tests verify this code

## Version Control

- [ ] Branch is clean (no uncommitted changes)
- [ ] Or changes are committed
- [ ] I know how to rollback if needed

## Scope

- [ ] Goal is ONE specific improvement
- [ ] I've broken it into <15 min steps
- [ ] I've identified all files to modify
- [ ] I have a plan for each step

## Risk Assessment

- [ ] Risk level identified (Low/Medium/High)
- [ ] Reason for assessment documented
- [ ] Mitigation plan for identified risks
